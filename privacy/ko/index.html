🔒 개인정보처리방침

최종 수정일: 2026년 2월 25일

1. 소개

BOTALKS("당사", "앱")는 이용자의 개인정보 보호를 중요하게 생각합니다. 본 개인정보처리방침은 BOTALKS AI 음성 대화 앱 사용 시 수집하는 데이터, 수집 방법, 사용 목적, 데이터를 수신하는 제3자에 대해 설명합니다.

2. BOTALKS 작동 방식

BOTALKS는 이용자가 직접 운영하는 OpenClaw Gateway에 연결되는 클라이언트 앱입니다. 당사는 이용자의 대화 데이터를 처리하는 중앙 서버를 운영하지 않습니다.

BOTALKS 사용 시 다음과 같은 데이터 흐름이 발생합니다:

  1. 음성 녹음: 마이크 버튼을 누르면 기기에서 음성이 녹음됩니다.
  2. 음성→텍스트 변환: 녹음된 오디오가 이용자의 OpenClaw Gateway로 전송되어 텍스트로 변환됩니다.
  3. AI 처리: 변환된 텍스트가 Gateway를 통해 이용자가 설정한 AI 제공업체(Anthropic Claude, OpenAI GPT 등)로 전달됩니다.
  4. 응답: AI가 생성한 텍스트 응답이 Gateway를 통해 돌아오고, 음성으로 변환되어 기기에서 재생됩니다.

3. 수집하는 데이터

3.1 음성 녹음 (오디오 데이터)

수집 항목: 음성 대화 기능 사용 시 이용자의 음성 녹음

수집 방법: 마이크 버튼을 누르고 있는 동안 기기 마이크로 녹음

전송 대상: 이용자 소유의 OpenClaw Gateway 서버 (음성→텍스트 변환 처리)

보존 기간: 텍스트 변환 후 즉시 폐기. 음성 녹음은 저장하지 않습니다.

3.2 텍스트 쿼리 (대화 데이터)

수집 항목: 음성에서 변환된 텍스트와 AI 응답

수집 방법: Gateway에서 음성→텍스트 처리 시 생성

전송 대상: Gateway에서 제3자 AI 제공업체(5조 참조)로 전송하여 응답 생성. 대화 내역은 화면 표시를 위해 기기에 로컬 저장될 수 있습니다.

보존 기간: 로컬 대화 내역은 앱 설정에서 삭제 가능. Gateway의 데이터는 이용자가 직접 관리합니다.

3.3 Gateway 연결 정보

수집 항목: Gateway URL 및 인증 토큰

수집 방법: QR 코드 스캔, 딥링크, 또는 수동 입력으로 이용자가 직접 입력

전송 대상: 이용자 기기에만 로컬 저장. 당사나 제3자에게 전송되지 않습니다.

3.4 구독 정보

수집 항목: 구독 상태 (활성/비활성, 플랜 유형)

수집 방법: Apple 인앱결제(StoreKit)를 통해 관리

전송 대상: Apple에서 처리. 당사는 구독 상태 확인만 받으며, 결제 정보는 받지 않습니다.

3.5 진단 데이터

수집 항목: 익명 충돌 보고서 및 기본 사용 분석

수집 방법: Expo/React Native 프레임워크에서 자동 수집

전송 대상: Expo(expo.dev)로 전송될 수 있음. 개인 식별이 불가능한 데이터입니다.

4. 수집하지 않는 데이터

  • 음성 녹음을 서버에 저장하지 않습니다
  • AI 대화 내용에 접근하지 않습니다
  • Gateway 인증 정보나 토큰을 수집하지 않습니다
  • 개인정보를 제3자에게 판매하지 않습니다
  • 광고, 프로파일링, 추적 목적으로 데이터를 사용하지 않습니다
  • 위치, 연락처 등 기기 데이터를 수집하지 않습니다

5. 제3자 서비스 및 데이터 공유

BOTALKS는 이용자의 자체 호스팅 OpenClaw Gateway에 연결되며, Gateway는 다음 제3자 AI 서비스 제공업체로 데이터를 전달할 수 있습니다:

서비스 제공업체 공유 데이터 목적
Anthropic (Claude) 변환된 텍스트 쿼리 AI 대화 응답 생성
OpenAI (GPT, Whisper) 변환된 텍스트, 오디오 (STT/TTS용) AI 응답, 음성 처리
기타 AI 제공업체 변환된 텍스트 쿼리 AI 대화 응답 생성
Apple (StoreKit) 구독 구매 데이터 인앱결제 처리
Expo (expo.dev) 익명 충돌 보고서 앱 안정성 모니터링

중요: 어떤 AI 제공업체가 이용자의 데이터를 수신하는지는 전적으로 이용자의 Gateway 설정에 따릅니다. 각 제공업체의 데이터 처리는 해당 업체의 개인정보처리방침에 따릅니다:

  • Anthropic 개인정보처리방침
  • OpenAI 개인정보처리방침

6. 앱 내 동의

Gateway나 제3자 서비스로 데이터가 전송되기 전에, BOTALKS는 데이터 및 개인정보 동의 화면을 표시하여:

  • 수집하는 데이터 (음성 녹음, 텍스트 쿼리)를 설명합니다
  • 데이터가 전송되는 대상 (이용자의 Gateway, 그리고 잠재적으로 AI 제공업체)을 명시합니다
  • 데이터 공유 전 이용자의 명시적 동의를 요구합니다

이용자는 앱 설정에서 언제든지 동의를 철회할 수 있으며, 이 경우 추가 데이터 전송이 중단됩니다.

7. 데이터 보안

당사는 다음과 같은 보안 조치를 시행합니다:

  • 앱과 Gateway 간 모든 통신은 암호화된 연결(TLS/HTTPS)을 사용합니다
  • 인증 토큰은 기기의 로컬 저장소에 안전하게 저장됩니다
  • 음성 녹음은 메모리에서 처리되며 영구 저장소에 기록되지 않습니다
  • 이용자 데이터는 당사 서버에 저장되지 않습니다 (중앙 서버 없음)

8. 이용자의 권리

  • 데이터 삭제: 앱을 삭제하여 로컬에 저장된 모든 데이터를 제거할 수 있습니다
  • 연결 초기화: 설정에서 언제든지 Gateway 연결을 초기화할 수 있습니다
  • 동의 철회: 설정에서 데이터 공유 동의를 철회할 수 있습니다
  • Gateway 관리: 이용자 소유의 OpenClaw Gateway에 있는 모든 데이터를 직접 관리합니다
  • 구독 관리: Apple App Store 설정에서 구독을 취소하거나 변경할 수 있습니다

9. 아동 개인정보

BOTALKS는 만 14세 미만 아동(개인정보보호법 기준)을 대상으로 하지 않습니다. 당사는 아동의 개인정보를 고의로 수집하지 않습니다.

10. 방침 변경

본 방침은 수시로 업데이트될 수 있습니다. 중요한 변경 사항은 앱을 통해 안내되며, 본 페이지 상단의 날짜가 업데이트됩니다.
